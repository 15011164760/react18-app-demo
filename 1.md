#### 2. 页面加载 js 脚本下载是同步的还是异步的，解析是按照什么顺序执行的

```
答案：1.当页面遇到脚本标签时，会暂停页面的渲染，先加载脚本，然后执行脚本，然后继续渲染 2.如果脚本设置async属性，那么脚本会异步加载，不会阻塞渲染，但是脚本加载完成之后，会阻塞渲染，等待脚本执行完成，然后继续渲染 3.如果脚本设置defer属性，脚本的执行会被延迟到整个文档都解析完成后（在 DOMContentLoaded 事件触发之前），并且脚本会按照它们在HTML中出现的顺序执行。因此，使用 defer 可以确保脚本不会阻塞页面的渲染，并且脚本的执行顺序得到保留。

```

#### 4. content-type 的值都有什么

```
答案：
text/plain：纯文本格式；
text/html：HTML格式；
text/css：Cascading Style Sheets；
text/javascript：JavaScript代码；
application/json：JSON格式数据；
application/xml：XML格式数据；
application/octet-stream：二进制流数据；
image/jpeg：JPEG格式图片；
image/gif：GIF格式图片；
image/png：PNG格式图片；
audio/mpeg：MP3格式音频；
video/mp4：MP4格式视频；
multipart/form-data：表单数据；
application/x-www-form-urlencoded：URL编码表单数据；
还有许多其他的 Content-type 类型，具体使用取决于需要传输的数据格式。
                        
```
